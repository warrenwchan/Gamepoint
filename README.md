# Gamepoint

How to use:
1. Install Meteor - curl https://install.meteor.com/ | sh

2. Install packages - npm i

3. Start App - meteor run

Default User Login:
user: john@john.com
password: john

Controls:
Q - Left Team Score +1
A - Left Team Score -1
E - Right Team Score +1
D - Right Team Score -1
SpaceBar - Team flip sides
